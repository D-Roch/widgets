Widgets Lift Module
==================

Provides:

* autocomplete
* calendars
* flot -- working with the [Flot Javascript plotting library](http://code.google.com/p/flot/).
* gravatar
* logchanger
* menu
* rssfeed
* sparklines
* tablesorter
* tree
* uploadprogress
